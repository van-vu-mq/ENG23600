>>>>> 3. AltSoftSerial Library

- Code libray used to control the UNO's BlueTooth LE module.

- Key information
	* Uses fixed pins assignment for RX and TX (RX-8, TX-9)
	* Asynchronous vs Arduino's synchronous Serial library 
	
- Download the library and add it to your Arduino Library folder. This is required.

https://www.pjrc.com/teensy/td_libs_AltSoftSerial.html
